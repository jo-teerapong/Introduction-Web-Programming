3. ให้หาว่า input type มีอะไรบ้างแต่ล่ะ type แตกต่างกันยังไง

1.Attribute

Tag Attribute How to use
Tag h header h1-h6(big to small) h1 word /h1
Tag p paragraph p Lorem ipsum dolor sit amet /p
Tag div เป็นแท็กที่ใช้เป็น box คลุมสิ่งอื่น div h1 I am div /h1 /div
Tag a เป็นคำสั่งเพื่อใช้ลืงค์ a href="ลิงค์ที่เราอยากใส่" ลิงค์ /a
Tag img คำสั่งเพื่อเพิ่มรูปภาพ img src="รูป" alt="#"
Tag ul เป็นคำสั่งที่ใช้ list แบบจุด ul li list1 /li li list2 /li /ul
Tag ol เป็นคำสั่งที่ใช้ list แบบตัวเลข ol li list1 /li li list2 /li /ol
Tag table เป็นคำสั่งไว้ฝช้สร้างตาราง table thead /thead tbody /tbody /table
Tag form element defines a form that is used to collect user input: form input... /form
Tag button A clickable button button type="button" Click Me! /button

2.differentiation

Tag name ความเหมือน ความต่าง
Tag hr เว้นบรรทัด ขึดเส้นใต้
Tag br เว้นบรรทัด ไม่ขีดเส้น
Tag ul ใช้ในการลิสต์ ลิสต์แบบจุด
Tag ol ใช้ในการลิสต์ ลิสต์แบบเลข
Tag select กดเพื่อเลือก กดที่เมนูและเข้าไปเลือก
Tag checkbox กดเพื่อเลือก เป็นกล่องสี่เหลี่ยมเล็กๆเลือกได้หลายอัน
Tag radio button กดเพื่อเลือก เป็นวงกลมเล็กๆเลือกได้แันเดียว
Tag div เป็นกล่องใช้คลุมสิ่งต่างๆ เป็น block
Tag span เป็นกล่องใช้คลุมสิ่งต่างๆ เป็น inline-block
Tag script การกำหนดข้อมูล ใช้ทางฝั่ง client
Tag style การกำหนดข้อมูล เป็นคำสั่งตกแต่งเว็บ

3.input type

type แตกต่าง
type text พิมข้อความ
type password พิมรหัสผ่านและมองไม่เห็น
type submit เป็นปุ่มกดเพื่อยอมรับ
type reset ปุ่มกดดเพื่อรีเซ็ต
type radio ปุ่มเลือกแบบเลือกได้แค่หนึ่ง
type checkbox ปุ่มเลือกแบบเลือกได้หลายอัน
type button เป็นปุ่มกด
type color เป็นปุ่มเลือกสี
type date เลือกวัน
type datetime-local เลือกวันและเวลา
type email ใส่ mail
type file เลือก file
type month เลือกเดือนและปี
type number เลือกตัวเลข
type Restrictions เลือกตัวเลขทีละเสต็ป
type Range เลือกขนาดของข้อมูลโดยการเลื่อน
type Search พิมข้อความแล้วกดค้นหา
type tel ใส่เบอร์
type time เลือกเวลา
type url ใส่ลิงค์
type week เลือกสัปดาห์และวันเดือนปี
